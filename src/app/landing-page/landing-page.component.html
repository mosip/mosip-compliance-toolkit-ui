Introducing the Compliance Tool Kit (CTK)

Ensure seamless integration with MOSIP standards
Compliance Tool Kit (CTK) is an online portal provided by MOSIP that allows partners to test the compliance of their products with MOSIP specifications. It supports three key components: SBI (Secure Biometric Interface), SDK (Biometric Service Providers), and ABIS (Automated Biometric Identification System).

Simplify Compliance with CTK

CTK supports compliance testing for these key specifications

SBI

- Ensures Secure Biometric Interface (SBI) compliance for unified communication and security protocols.

Android SBI

- Android SBI is developed to ensure consistent and secure communication between biometric devices and Android applications, Android SBI follows the same principles as the broader SBI specification.

- CTK provides a compliance testing toolkit specifically designed for Android SBI implementations

Compliance Toolkit APK

Biometric SDKs

- Verifies iBioAPI compliance for quality check, match, extraction, and conversion functions.

- Enables testing of independent HTTP service integration.

ABIS

- Guarantees 1:N deduplication accuracy through ABIS API compliance.

- Validates JSON message format for seamless communication.
